# Customer Segmentation with K-Means

This project performs **customer segmentation** using the **K-Means algorithm** on the
Kaggle dataset:

> [Customer Segmentation Tutorial in Python](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)

---

## Dataset Description

The dataset contains 200 customer records with the following columns:

- **CustomerID**
- **Gender**
- **Age**
- **Annual Income (k$)**
- **Spending Score (1-100)**

The *Spending Score* quantifies a customer's purchase behavior:  
a high score indicates a strong willingness to buy.
